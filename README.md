# BeeHub - Repositório de Documentos

## Descrição 
O BeeHub é um sistema que veio para facilitar a comunicação entre o aluno e professor, organização e manuntenção de TCCs para pessoas que possuem dificuldades em realizá-los.

## Como rodar o projeto
## Ferramentas
 - Java
 - React
 - Spring Boot
 - JPA
 - Hibernate
 - MySQL
